# IPinfo
IPinfo是一个整合多接口的IP地址查询工具

### 主要功能
* 自动获取内网IP、公网IP
* 支持多个数据接口
* 支持一键查询所有IP接口，方便对比数据

### 使用
IPinfo使用PHP开发，开箱即用，下载源码后直接放到站点根目录即可。
